# Jinjoo Travel - 가톨릭 성지순례 전문 여행사

## 프로젝트 소개
가톨릭 성지순례 전문 여행사 웹사이트입니다. React와 TypeScript를 사용하여 개발되었으며, 
성지순례 상품 관리, 예약 시스템, 성지 정보 제공 등의 기능을 제공합니다.

## 주요 기능
- 성지순례 상품 조회/등록/수정
- 성지 정보 제공
- 관리자 기능
- 예약 시스템
- 이미지 갤러리

## 기술 스택
- React
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- Lucide React

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
